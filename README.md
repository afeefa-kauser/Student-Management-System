# Student Management System

## Project Overview

This project is a SQL-based Student Management System designed to manage student records and perform database operations efficiently.

## Features

- Store student information
- Add student records
- Update student details
- Delete student records
- Search student data
- Generate reports using SQL queries

## Database Structure

Tables included:

### Students
- Student_ID
- Student_Name
- Age
- Department
- Email

### Courses
- Course_ID
- Course_Name

### Enrollment
- Enrollment_ID
- Student_ID
- Course_ID

## SQL Concepts Used

- CREATE TABLE
- INSERT
- SELECT
- UPDATE
- DELETE
- WHERE
- ORDER BY
- GROUP BY
- JOIN
